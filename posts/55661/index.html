《末日生存手游【守望黎明】 一键端+ LINUX手工外网端+外网教程》正式版

📌 游戏介绍

一键端教程下载里有 VM的就不讲了
用到的工具：
finalshell（链接服务器用的）
N++（文本修改工具）
UE（修改客户端文件）注意：准备好一个13位的IP或者域名（修改客户端会用到）必须13位的IP或者域名
测试机器：4H8G5M
系统： CentOS 7.2 64位
低于485的没测过，有问题自己搞定
安装宝塔
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
[color=rgb(51, 102, 153) !important]复制代码
安装 Nginx1.16、MySQL5.6、PHP5.6、phpmyadmin（这个东西是连数据库用的，不用开远程）
php安装扩展redis
关闭防火墙
systemctl stop firewalld.service
systemctl disable firewalld.service
[color=rgb(51, 102, 153) !important]复制代码
放行端口1:65535
上传0.zip到根目录
解压
cd /
unzip 0.zip
[color=rgb(51, 102, 153) !important]复制代码
安装java
cd /52gmsy/down
tar -zxvf jdk-8u11-linux-x64.tar.gz
[color=rgb(51, 102, 153) !important]复制代码
修改配置文件
/etc/profile
[color=rgb(51, 102, 153) !important]复制代码
在最下面加入
export JAVA_HOME=/52gmsy/down/jdk1.8.0_11
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib:$CLASSPATH
export JAVA_PATH=${JAVA_HOME}/bin:${JRE_HOME}/bin
export PATH=$PATH:${JAVA_PATH}
贴到最底部就行
及时生效
source /etc/profile
创建网站 端口8081
指向目录 /www/wwwroot/swlm
关闭跨站
更改nginx默认端口
/www/server/panel/vhost/nginx/phpfpm_status.conf 把里面的listen 80; 改成 listen 5252; 把里面的unix:/tmp/php-cgi-56.sock; 改成 127.0.0.1:9000;
/www/server/panel/vhost/nginx/0.default.conf 把里面的listen 80; 改成 listen 5252;
/www/server/nginx/conf/enable-php-56.conf 把里面的unix:/tmp/php-cgi-56.sock; 改成 127.0.0.1:9000;
/www/server/nginx/conf/enable-php.conf 把里面的unix:/tmp/php-cgi-56.sock; 改成 127.0.0.1:9000;
/www/server/php/56/etc/php-fpm.conf 把里面的listen = unix:/tmp/php-cgi-56.sock; 改成 listen = 127.0.0.1:9000
重启fpm+nginx
sudo service nginx restart
sudo service php-fpm restart
这里可以先查一下端口有没有错误
查看端口命令
netstat -ltnp
注意不能有80端口
服务端IP修改
/52gmsy/SFS2X/extensions/COK2/resource/servers.xml
给权限
chmod -R 777 /52gmsy
修改数据库密码为123321
创建数据库和输入下面命令自动导入数据
mysql -uroot -p123321
create database cokdb_global;
create database cokdb2;
create database domain_db;
create database cokdb_ingamemail;
create database cokdb_record;
create database cokdb_template;
create database warfaredb;
use cokdb_global;
source /52gmsy/down/cokdb_global.sql;
source /52gmsy/down/server_info.sql;
source /52gmsy/down/cokdb_global_serverlist.sql
use cokdb2;
source /52gmsy/down/cokdb2.sql;
use domain_db;
source /52gmsy/down/domain_db.sql;
use cokdb_ingamemail;
source /52gmsy/down/cokdb_ingamemail.sql;
use cokdb_record;
source /52gmsy/down/cokdb_record.sql;
use cokdb_template;
source /52gmsy/down/cokdb_template.sql;
use warfaredb;
source /52gmsy/down/warfaredb.sql;
flush privileges;
结束后输入exit
游戏启动
cd /52gmsy/apache-apollo-1.7.1/bin/mybroker/bin
./apollo-broker-service start

游戏停止
cd /52gmsy/apache-apollo-1.7.1/bin/mybroker/bin
./apollo-broker-service stop
[color=rgb(51, 102, 153) !important]复制代码
后台启动
cd /52gmsy/SFS2X
./sfs2x-service start
[color=rgb(51, 102, 153) !important]复制代码
后台停止
cd /52gmsy/SFS2X
./sfs2x-service stop
[color=rgb(51, 102, 153) !important]复制代码
第一次启动有点久
耐心等待 大概十几分钟到几十分钟不等
可以通过查询端口看看9933端口是否启动来判断
查看端口命令
netstat -ltnp
前台运行
./sfs2x-service start-launchd （要是运行这个命令，关闭软件的话游戏就进不去了，一般用来调试查看比较方便，可以看到很多输出的信息）
客户端IP修改（UE软件修改，ASCII要打勾，直接全部替换就行，位数要13位，包括点）
lib\armeabi\libgame.so 替换122.51.133.53
修改完用好压打开APK，把so文件拖到对应的位置就行了
然后APK签名

选择testam—GAME UID填1—–点change
以后登陆直接点最顶上的continue
