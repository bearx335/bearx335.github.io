《春秋Q传/一键端/注册站/客户端/GM模式/教程》正式版

📌 游戏介绍

0709（最终版）
·镇国之战：每天19：50系统发公告，20：00开启。
大/小Boss刷新时，游戏画面的公告中加入了Boss刷新时的坐标信息，但是该坐标为4位数，不知道对应到小地图3位数坐标的换算公式。有兴趣的自己研究吧。
调整了事件中的收益倍数。
杀死大/小Boss后会刷怪的数量由原版的80（小Boss），100（大Boss）都改为了0个。小怪太多影响刷Boss了，就改了。
·橙色武器：原版爆率文件中没有设定橙色武器的掉率，也没有任何任务可以获得，应该是没有设置获取途径。
杀死镇国之战中的Boss，有概率获得镇国之宝，交给周王后可随机换绿色（25%概率）或紫色（75%概率）装备。将原版的兑换绿色/紫色装备，改为了兑换紫色/橙色装备，概率各50%。0706关于游戏版本：通过程序和不同的设定文件内容确认，该单机为70级版本。比如，主线任务设置文件没有70级以后的，强行修改代码激活反恐任务70+任务时出错等。
·服务端启动/关闭：启动命令中加入路径，防止由于目录错误导致的启动/关闭失败。
启动命令
cd /root/chq-docker
docker compose up -d –force-recreate
关闭命令
cd /root/chq-docker
docker compose stop
·注册页面：用ChatGPT重新写了一个美化了的注册页面，并把原版的代码Bug改了。
·每天能获取的经验上限：全等级改为原版的10000倍。
·春秋币上限：改为10亿。商城买【小金库】使用后激活。※小金库的文字描述还是四千万
·生活技能：甲，纹，印的生产材料中的魂需求数，全等级改为1个（普通材料和花费没改）。各种甲，纹，印的产出数量改为20个。
·镶嵌：甲，纹，印的镶嵌成功率改为100%。按照装备的等级区段和品质分别将镶嵌的所有属性上限系数改为15-17倍左右（百分比属性的上限应该不会超过100%）。
原版中随着成功镶嵌次数的增加，新成功镶嵌上的属性值会衰减，最低为1。将原版该衰减机制取消，每次成功镶嵌时增加的属性值在1-100的范围随机（百分比属性也适用）。
蓝色，绿色，紫色，橙色装备的镶嵌系数分别为修改后的15，20，25，30。举例：蓝色60+护甲类镶嵌【物理闪避】，属性上限值由原版88->1230。 ※大概测了几个都没问题。
关于镶嵌上限确认，我在代码中加入了实时显示当前属性状态的日志输入，在游戏画面中的成功镶嵌时的系统通知中显示，并用颜**分当前值和上限值。
（本来没打算改镶嵌，成长属性上限1000的问题没有好的解决方案，作为补偿方案把镶嵌增强了。简单测试了下，镶嵌的属性值可以突破1000。但是属性值过大是否会发生负增长就不知道了，没测试）
·装备改造：成功率改为100%。 ※跟镶嵌的计算逻辑通用，没测试0703
·新注册账号送10亿金币（珍宝阁商城用的货币）。
·吕不韦任务：奖励经验10倍，奖励金钱10倍。
·霸主任务：奖励经验10倍，奖励金钱10倍。
改为单人可接取，需单人组队。
每天接取任务数上限由5次改为100次。
收集任务的各种材料都改为1个（蓝色以上装备原本就是提交1个，没改）。 ※没碰到材料的收集任务，没测试。
·人物升级所需经验：原版的1/10。客户端游戏界面上的数值还是原版。
·元神升级所需经验：原版的1/10。客户端游戏界面上的数值还是原版。
·打坐经验：100倍（默认20级+生效，保持原版设定）
·体力恢复：10倍（包括平时和打坐）
·人物每集成长属性：全职业的力量，体制，灵巧，敏捷，智力，精神全部改为原版的10倍。
·专精点数：每级一级提升的专精点数由1点改为5点。保持原版30级+开始获取的设定。
·装备爆率：所有非Boss怪改为了原版的100倍爆率。Boss改为了100%爆率。
·抓元神：抓取成功率保持原版，成功抓取的话100%随机出6属性并且满资质。元神成长度保持原版。
·生活职业：单人物可学所有生活技能（第一个生活职业需要先成为该职业，之后的就不需要成为那个职业了，从NPC选择学习职业技能就行）
·制造生活物品：材料和花费没改，物品的产出数量调整为原版的2-10倍不等。大部分常用物品的产量都设为了20。（没钱了就做霸主）
·绿色品质碎片合成：25个碎片合成5个绿色品质石，改为合成为25个绿色品质石。客户端界面的合成数量还是原版数值。
·紫色品质碎片合成：25个碎片合成1个紫色品质石，改为合成为25个紫色品质石。客户端界面的合成数量还是原版数值。
·装备品质升星：蓝色→绿色→紫色的升星成功率都改为了100%，升星时用1块品质石即可。客户端界面的成功率同步显示 100%。
·装备强化：成功率改为了100%，强化时用1块宝石即可。
·家族创建：创建家族的费用由200万改为1春秋币，各种材料数量也全改为了1个。
创建家族时的组队最低人数改为了1人。保持原版最低40级的设定。
·家族维护设置：家族人数不足，3天后解散改为3650天。家族人气为持续7天为0时解散改为3650天。家族人数低于3人时解散改为1人。没提到的就是没改。 ※没测试·周王宝藏：开放时间改为每个小时的00分45分都可进入，每个小时的45分60分这15分钟保持原版的不可进入。
改为单人可进，需单人组队。副本中解散组队的话，仍然会被传送出。
进入时的家族贡献消耗，全等级都改为1，消耗的春秋币保持原版。
原版的5碎片兑换一本书，改为2个碎片合成。
无论有没有家族，每天进入最大次数由原版的1-2次，改为100次。（应该够了吧！）
进入周王宝藏前后两次要求的间隔时间由5分钟改为0分钟。（每个小时的00分~45分之间可以随意进出。当然，每天的总次数受限于100次）
上述设定相关的文字说明也在客户端界面上全部同步修改了。
·反恐任务：改为单人可接，需单人组队。
奖励经验和金钱都改为原版的10倍。（40级任务杀完一轮合计大概也有500w+经验，200w+春秋币了吧。但是考虑到杀怪效率，其实和做几次霸主花费的时间和收益差不多。做不做看个人喜好了）
·试炼堂：活动举行的星期由1-5，改为每天举行。
进入时的家族贡献消耗，全等级都改为1，消耗的春秋币保持原版。 ※游戏画面上提示扣100贡献，实际上扣1贡献。
无论有没有家族，每天进入最大次数改为100次。
调整杀怪个数和经验加成系数，杀怪数基数由300改为30，经验加成系数改为原版的10倍。杀怪数超过210个，经验加成系数固定为26，不再增长。（获取经验的主要途径本来也不是靠这副本，全当是一点福利了）
杀怪超过300，可以领取奖励宝石的设定改为超过即可30领取。
调整根据杀怪数等级给与宝石的设定，杀怪数基数由300改为30，每增加30杀怪数给与的宝石数量保持原版（强化装备已经100%成功率了，不需要过多宝石了）。杀怪数超过210个后能换取宝石的数量不再增长。
杀怪掉落的品质宝石几率保持原版。原版已经不低了，就没改。
活动开始时间还是每2小时一次，每次开放1小时活动时间的设定保持原版没改。 ※如果设置每小时开启的话，活动时间就要设置45分钟了，需要给下一场提前预留10分钟的入场准备时间。不然程序会出错，亲自验证。
·上古遗迹：由原版隔周的周六和周日各1次，改为每天的0-23点一直开放。反正就是服务器启动之后，随时都能进。NPC的游戏画面关于进入时间的描述还是原版，这个估计在客户端.pak里，暂时改不了。 ※结束时间设置24点的话有BUG风险，没时间测试就这样吧。
每一分钟经验值增长的倍数改为原版的10倍，并配合调整每分钟可获得的最大经验值为10倍。
每分钟家族贡献增长的倍数改为原版的10倍，并配合调整每分钟可获得的最大家族贡献为10倍。
宝箱生成时间间隔由5分钟改为1分钟。每一等级宝箱的最大个数默认是15个上限，保持原版。
10个碎片换紫装改为2个碎片即可兑换。
（缺贡献了就去上古遗迹，每分钟奖励600家族贡献）
